## Environment Variables

### Running Acceptance Tests

```bash
TF_ACC=1 go test -count=1 -v ./... -timeout 30m
```

### Provider Configuration (required)

| Variable | Purpose |
|----------|---------|
| `AAP_HOSTNAME` | AAP server URL (preferred). Fallback: `AAP_HOST` |
| `AAP_USERNAME` | Basic auth username (required if no token) |
| `AAP_PASSWORD` | Basic auth password (required if no token) |
| `AAP_TOKEN` | Token auth (mutually exclusive with username/password) |
| `AAP_INSECURE_SKIP_VERIFY` | Skip TLS verification (default: `false`) |
| `AAP_TIMEOUT` | HTTP request timeout in seconds (default: `5`) |
| `TF_ACC` | Set to `1` to enable acceptance tests |

### Test Resource IDs (generated by `testing/setup-env.sh`)

| Variable | Purpose |
|----------|---------|
| `AAP_TEST_JOB_TEMPLATE_ID` | Basic job template |
| `AAP_TEST_JOB_TEMPLATE_INVENTORY_PROMPT_ID` | Job template with inventory prompt-on-launch |
| `AAP_TEST_JOB_TEMPLATE_ALL_FIELDS_PROMPT_ID` | Job template with all promptable fields |
| `AAP_TEST_WORKFLOW_JOB_TEMPLATE_ID` | Basic workflow job template |
| `AAP_TEST_WORKFLOW_JOB_TEMPLATE_FAIL_ID` | Workflow template with failure node |
| `AAP_TEST_ORGANIZATION_ID` | Non-default organization |
| `AAP_TEST_WORKFLOW_INVENTORY_ID` | Inventory for workflow testing |
| `AAP_TEST_INVENTORY_FOR_WF_ID` | Additional inventory for workflow tests |
| `AAP_TEST_JOB_FOR_HOST_RETRY_ID` | Long-running job template for host delete-retry |
| `AAP_TEST_JOB_TEMPLATE_FAIL_ID` | Job template with failing playbook |
| `AAP_TEST_DEMO_CREDENTIAL_ID` | Machine credential for launch tests |
| `AAP_TEST_LABEL_ID` | Label for all-fields-on-prompt tests |
| `AAP_TEST_DEFAULT_INSTANCE_GROUP_ID` | Default instance group |

### Ansible Provisioning (derived in `testing/setup-env.sh`)

| Variable | Source |
|----------|--------|
| `CONTROLLER_HOST` | From `AAP_HOSTNAME` |
| `CONTROLLER_USERNAME` | From `AAP_USERNAME` |
| `CONTROLLER_PASSWORD` | From `AAP_PASSWORD` |
| `CONTROLLER_VERIFY_SSL` | Inverse of `AAP_INSECURE_SKIP_VERIFY` |

| `AAP_AUTOMATION_HUB_TOKEN` | Required in `testing/.env` if collections not installed |
